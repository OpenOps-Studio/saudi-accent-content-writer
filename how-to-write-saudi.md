## 1. Saudi Dialect Fundamentals

### 1.1 Three-Tier Dialect System

#### Tier 1: Saudi Formal (الفصيحة السعودية)
**When to use**: Government, institutional, formal announcements

**Characteristics**:
- 80% Modern Standard Arabic (MSA) + 20% Saudi dialect
- Professional vocabulary
- Formal sentence structures
- Occasional Saudi flavor words
- No slang or colloquialisms

**Example**:
```
✅ "نسعد بتقديم منتجنا الجديد الذي يعكس التزامنا بتطوير الخدمات المحلية"
   (We are delighted to present our new product which reflects our commitment 
    to developing local services)

❌ "كيفك يا رايق، عندنا منتج ما بتاعك الأول"
   (Hey buddy, we got a product you've never seen before) - Too casual
```

#### Tier 2: Saudi Conversational (السعودية المحكية)
**When to use**: Social media, consumer content, brand personality

**Characteristics**:
- 70% Saudi dialect + 30% MSA
- Natural, approachable tone
- Common Saudi slang expressions
- Conversational sentence flow
- Emotional warmth and connection

**Example**:
```
✅ "مرحبا يا حلوين! المنتج هذا غيّر حياة الناس، وبصراحة جداً واجد مميز"
   (Hey everyone! This product changed people's lives, and honestly it's really special)

❌ "السلام عليكم ورحمة الله وبركاته، يود الجميع اعلام كافة أن..."
   (Peace be upon you... we would like to inform everyone...) - Too formal for social
```

#### Tier 3: Gulf Regional (الخليجية)
**When to use**: Cross-GCC campaigns, regional expansion

**Characteristics**:
- 50% Saudi + 30% Gulf (UAE/Kuwait/Bahrain) + 20% MSA
- Incorporates Gulf vocabulary
- Maintains Saudi authenticity
- Accessible to whole GCC region

**Example**:
```
✅ "ما تخلي الفرصة تروح! المنتج يبدي روعة ويطلع بنتايج حلوة"
   (Don't miss the opportunity! The product is amazing and delivers great results)
```

### 1.2 Key Saudi Dialect Features

#### Pronunciation & Written Representation

| Feature | Example | Notes |
|---------|---------|-------|
| **Dropped letters** | "تبدي" instead of "تبدو" | In casual writing |
| **Emphatics** | ص، ض، ط، ظ used naturally | Maintain authenticity |
| **Colloquial particles** | "لا" (not), "شنو" (what) | Context-dependent |
| **Negation** | "ما" instead of "لا" | Saudi standard negation |
| **Time markers** | "الحين" (now), "بعدين" (later) | Temporal expressions |

#### Common Saudi Expressions (و لهجة)

| Expression | English | Context | Authenticity |
|------------|---------|---------|--------------|
| "واجد" | Very/Really | "واجد حلو" = really nice | ⭐⭐⭐⭐⭐ |
| "ما بالك" | Why/What's wrong | "ما بالك قاعد وحدك" | ⭐⭐⭐⭐⭐ |
| "يبدي" | Seems/appears | "هذا يبدي حلو" = this looks nice | ⭐⭐⭐⭐⭐ |
| "خلاص" | That's it/Done | "خلاص انتهينا" = we're done | ⭐⭐⭐⭐⭐ |
| "روعة" | Awesome/Cool | "المنتج روعة" = the product is awesome | ⭐⭐⭐⭐⭐ |
| "راقت لك" | Appeal to you | "هل راقت لك الفكرة" = do you like the idea | ⭐⭐⭐⭐ |
| "يقطين" | You're late | Regional/tribal context | ⭐⭐⭐ |
| "يا جماعة" | Guys/folks | "يا جماعة انتبهوا" = everyone pay attention | ⭐⭐⭐⭐⭐ |

## 2. Tone & Voice Framework

### 2.1 Tone Matrix by Platform

```
┌──────────┬──────────┬──────────────┬──────────────┬──────────┐
│ Platform │ Formality│ Warmth       │ Humor        │ Emoji    │
├──────────┼──────────┼──────────────┼──────────────┼──────────┤
│ LinkedIn │ 8/10     │ 5/10         │ Minimal      │ 1-2 only │
│ Instagram│ 4/10     │ 8/10         │ Playful      │ 3-5      │
│ TikTok   │ 2/10     │ 9/10         │ Very playful │ 5-7      │
│ Twitter  │ 6/10     │ 6/10         │ Occasional   │ 1-3      │
│ Corporate│ 9/10     │ 3/10         │ None         │ None     │
└──────────┴──────────┴──────────────┴──────────────┴──────────┘
```

### 2.2 Sentence Structure Patterns

#### LinkedIn (Professional)
```
Pattern 1: [Observation] + [Insight] + [Call to Action]
"لاحظنا أن العملاء يبحثون عن حلول حقيقية. لذلك طورنا منتجاً يفهم احتياجاتهم. 
هل تريد أن تعرف المزيد؟"

Pattern 2: [Question] + [Insight] + [Position]
"كم مرة شعرت أن الخدمات الحالية لا تلبي احتياجاتك؟ 
في شركتنا، فهمنا هذا التحدي، وركزنا على تقديم أفضل الحلول."
```

#### Instagram (Engaging)
```
Pattern 1: [Hook Emoji] + [Emotion] + [Brief Value] + [CTA]
"✨ أحنا مشتاقين نشاركك شيء جديد! منتج اللي ما بتتوقعه، وبصراحة يبدي واجد حلو! 
Swipe up 👆"

Pattern 2: [Direct Address] + [Story] + [Tag]
"يا حلوين، قصتنا بدأت من الحاجة البسيطة... 
وأصبحت أكبر حل بالسوق! هذا بفضلكم انتو 🙏💚"
```

#### TikTok (Viral)
```
Pattern 1: [Hook in first 3 seconds] + [Story] + [Surprise/Payoff]
"يالا عندك 3 ثواني بس... كم منتج سعودي بتستخدم الحين؟ 
(تصوير سريع لـ 4-5 منتجات)
لما تقول: الي أبيه، هذا! 😂"

Pattern 2: [Relatable Scenario] + [Problem Solve] + [Tag]
"كل يوم أصحيك الشمس في العين 😂 
تال ما بعد كذا مع... [منتج] 
التاغ الاصدقاء الي يعانون مثلك 👇"
```

## 3. Cultural Authenticity Guide

### 3.1 Core Saudi Values (العادات والتقاليد)

#### Value 1: Family & Relationships (الأسرة)
```
✅ Culturally resonant approach:
"هذا المنتج يجمع الأسرة، ويخلي الوقت مع أحبايبك أجمل"

❌ Avoid:
"هذا المنتج للأفراد اللي ما عندهم وقت" - Dismisses family
"استخدمه بنفسك بعيد عن الناس" - Isolating message
```

#### Value 2: Respect & Authority (الاحترام والسلطة)
```
✅ Culturally resonant approach:
"بفضل توجيهات قيادتنا الحكيمة، طورنا هذا المنتج بما يتماشى مع رؤية 2030"

❌ Avoid:
"نحن نعارض سياسات الحكومة" - Directly challenging authority
"غير مهم ما قالوا، رأينا أفضل" - Disrespectful tone
```

#### Value 3: Trust & Loyalty (الثقة والولاء)
```
✅ Culturally resonant approach:
"زي ما تثقون فينا منذ سنوات، ما بنخيبكم هالمرة"

❌ Avoid:
"جرب منتجنا، بعدين شوف اللي أحسن" - No brand loyalty
"أحنا أفضل من الكل والنهاية" - Arrogant positioning
```

#### Value 4: Authenticity & Integrity (الصراحة والنزاهة)
```
✅ Culturally resonant approach:
"بصراحة تامة، في تحديات لازم نواجهها معاً، وحنا اللي نحلها"

❌ Avoid:
"هذا المنتج كامل مثالي، ما فيه مشاكل" - Unrealistic claims
"لا نقول لك الحقيقة، بس اشتر" - Deceptive marketing
```

---

### 3.2 Religious & Cultural Sensitivity

#### What's Appropriate ✅
```
• References to Islamic values in business context
• Respect for prayer times and Ramadan considerations
• Alignment with Saudi national pride
• Respect for the monarchy
• Values of hard work and community benefit

Examples:
"بفضل الله ثم جهودنا، وصلنا لهنا"
"في شهر رمضان الكريم، أنشاء تشارك خيرك"
"بفضل قيادتنا الحكيمة وهمة أبنائها"
```

#### What to Avoid ❌
```
• Direct religious preaching
• Political criticism
• Tribal references or divisions
• Mocking religious practices
• Making unevidenced religious claims

Examples to NEVER use:
"يا كافر، اشتري المنتج" - Offensive religious language
"الحكومة بتظلمنا" - Political criticism
"بني فلان أفضل من بني علان" - Tribal divisions
"هذا حرام، فالمنتج حلال" - Unsubstantiated religious claims
```

---

### 3.3 Cringe Factor Elimination

#### The "Cringe Test": Would You Say This Out Loud to a Saudi Friend?

| Content | Cringe Score | Why | Fix |
|---------|--------------|-----|-----|
| "يا جماعة الخير، منتجنا يبدي واجد حلو!" | ✅ 0/10 | Natural, friendly | - |
| "انت ما بتبي تكون خاسر، صح؟" | ⚠️ 6/10 | Manipulative pressure | "جرب التجربة المجانية" |
| "يالا الآن أو ندمان للأبد!" | ❌ 9/10 | Too aggressive | "متاح الآن للطلب" |
| "وش رايك نصير أحسن إصدخاء؟" | ❌ 8/10 | Fake intimacy | "نتطلع لرأيك" |
| "لولا الكسل، كنت استخدمت كل المنتجات" | ⚠️ 7/10 | Self-deprecating | Focus on benefits |

## 4. Platform-Specific Guidelines

### 4.1 LinkedIn (B2B & Professional)

**Voice**: Expert, Thought Leader, Trustworthy  
**Length**: 150-300 words typical  
**Engagement**: Comments and Shares > Likes

**Saudi-specific tips**:
```
✅ DO:
• Reference Vision 2030 alignment
• Showcase company leadership and stability
• Share business insights with Saudi market context
• Respect formal business culture
• Use professional Arabic terms

❌ DON'T:
• Use excessive emoji (max 1-2)
• Be too casual or joking
• Make controversial statements
• Use TikTok-style language
• Overstate achievements
```

**Content Formula**:
```
[Professional Hook] + [Business Insight] + [Vision 2030 Alignment] + [Call to Action]

Example:
"في حقبة التحول الرقمي التي نعيشها، الشركات السعودية تحتاج لشركاء موثوقين.
لذلك أطلقنا منصتنا الجديدة التي تساهم في تحقيق أهداف رؤية 2030.
[Details about how it aligns with vision]
من يريد أن يعرف المزيد؟ أضغط في التعليقات 👇"
```

---

### 4.2 Instagram (B2C & Brand Building)

**Voice**: Friendly, Approachable, Culturally Connected  
**Length**: 80-150 words (caption)  
**Engagement**: Likes, Comments, Story Shares

**Saudi-specific tips**:
```
✅ DO:
• Use warm, friendly Saudi dialect
• Showcase team and company culture
• Celebrate Saudi heritage and identity
• Use relevant emoji (3-5)
• Tell stories that resonate with values

❌ DON'T:
• Use MSA (too formal for Instagram)
• Be overly salesy
• Ignore brand consistency
• Use inappropriate slang
• Post without visual content
```

**Hashtag Strategy (Saudi Context)**:
```
Branded: #اسم_الشركة #حقنا_سعودي
Reach: #سعودي #تسوق #عروض
Niche: #دعم_محلي #منتجات_سعودية #ريادة_أعمال
Seasonal: #رمضان_عطاء #العيد_معنا #الوطن_العزيز
```

**Story Post Formula**:
```
Slide 1: Hook/question
"شنو أكثر شيء ما بتركوه في البيت؟ ⬇️"

Slide 2: Show product in action
[Video/image of product being used]

Slide 3: CTA
"اطلبوه الحين من الرابط في البايو ✨"
```

---

### 4.3 TikTok (Viral & Entertainment)

**Voice**: Fun, Casual, Authentic, Relatable  
**Length**: 15-60 seconds  
**Engagement**: Views, Shares, Comments = Algorithm boost

**Saudi-specific tips**:
```
✅ DO:
• Start with hook in first 1-2 seconds
• Show real people, not just products
• Use trending Saudi music/sounds
• Be authentic and slightly imperfect
• React to trends quickly
• Use Saudi humor (light-hearted)

❌ DON'T:
• Be corporate or stiff
• Use MSA or formal language
• Ignore trending audio
• Post low-quality video
• Be offensive or controversial
• Force the product placement
```

**Hook Formulas**:
```
Hook 1: Relatable Problem
"كل شخص سعودي بيشتكي من..."
[Show pain point]
"الحل هنا 👇"

Hook 2: Surprising Fact
"ما بتتوقع هذا الشيء يصير بـ ٣ ثواني!"
[Demo product]

Hook 3: Question
"أي واحد يبي يختبر هذا معايا؟"
[User-generated content style]
```

### 4.4 Twitter/X (News & Engagement)

**Voice**: Quick, Witty, Newsworthy  
**Length**: 280 characters (use efficiently)  
**Engagement**: Retweets, Quote tweets, Replies

**Saudi-specific tips**:
```
✅ DO:
• Post timely news and updates
• Respond quickly to mentions
• Use humor when appropriate
• Engage in conversations
• Share company news and milestones

❌ DON'T:
• Get involved in political debates
• Respond to controversial topics
• Use slang that might offend
• Post misinformation
• Engage in Twitter wars
```

**Tweet Formulas**:
```
Formula 1: News + Emoji
"🚀 فقط أطلقنا منتجنا الجديد! المتاح الآن عند..."

Formula 2: Question + Context
"كم من المرات شعرت أنك بحاجة لـ [solution]؟
نحن أطلقنا الحل اليوم 🎯"

Formula 3: Retweet + Comment
"[Retweet of industry news]
في شركتنا نفكر بـ [aligned perspective]"
```

## 6. Common Mistakes to Avoid

### Mistake 1: Forced Slang
```
❌ "واجد، واجد، واجد! هذا يبدي يبدي روعة! يالا بسرعة!"
   (Overusing slang words makes it feel fake)

✅ "هذا المنتج يبدي واجد حلو، وفي أسعار تناسب الجميع"
   (Natural integration of 1-2 slang words)
```

### Mistake 2: Inauthentic Tone
```
❌ "يا أحبائي الأعزاء، تشرفوا بقبول منتجنا المتميز!"
   (Too formal, doesn't match Saudi conversational style)

✅ "يا حلوين، بنقدم لكم المنتج الي كنتو تبون"
   (Warm, authentic, conversational)
```

### Mistake 3: Cultural Insensitivity
```
❌ "هذا حرام، فمنتجنا حلال" (Making unsubstantiated religious claims)
❌ "ما تقول لحد أنك بتستخدم هذا" (Shame-based marketing)
❌ References to alcohol, pork, or other sensitive topics

✅ Focus on universal values: family, trust, quality, progress
```

### Mistake 4: Cringe Humor
```
❌ "هههههه، كل واحد فيكم فاشل لولا منتجنا!" (Offensive)
❌ "يا كسول، الحين وش رايك تشتري؟" (Insulting)

✅ "تعبت من نفس الأشياء؟ جرب شيء جديد" (Light, inviting)
```

### Mistake 5: Ignoring Platform Norms
```
❌ Posting LinkedIn-style formal text on TikTok
❌ Using emoji-heavy Instagram style on LinkedIn
❌ Copying international marketing without Saudi adaptation

✅ Adapt tone, length, and style to platform norms
```

### Mistake 6: Vague Value Propositions
```
❌ "منتج جديد وعظيم!"
   (What makes it great? Why should I care?)

✅ "توفر وقتك 50% يوميًا، وتكلفتها أقل من المنافسين بـ 30%"
   (Specific, quantified benefits)
```

## Quick Reference Checklist

Before publishing any Saudi content:

- [ ] Does it sound natural when read out loud in Saudi Arabic?
- [ ] Are all claims factual and substantiated?
- [ ] Does it align with Saudi values (family, trust, progress)?
- [ ] Is the platform tone appropriate? (LinkedIn ≠ TikTok)
- [ ] Does it avoid religious/political sensitivity issues?
- [ ] Is the CTA clear and easy to understand?
- [ ] Have I used authentic dialect, not forced slang?
- [ ] Would a Saudi friend say "that's cringe" about this?
- [ ] Is it compliant with CITC and Media Ministry standards?
- [ ] Does it advance the business objective?
