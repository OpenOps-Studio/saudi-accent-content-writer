## STEP 2: Select Your Execution Mode

Based on your needs, choose one:

### MODE 1: Express Launch (30 minutes)
**Use when**: Quick content needed, time-sensitive
**Includes**: 
- Nodes: PR-01, PR-05, PR-08, PR-10, PR-12
- Outputs: Content variants + quick testing plan
- Deliverables: Ready-to-launch content (no strategy deep-dive)

### MODE 2: Full Strategic Run (120 minutes)
**Use when**: Major campaign, starting from scratch
**Includes**:
- All 13 processing nodes
- All 7 output nodes
- Complete strategy + content + testing framework

### MODE 3: Institutional Content Sprint (90 minutes)
**Use when**: B2B/government communication
**Includes**:
- Full processing nodes with emphasis on PR-07 (institutional positioning)

## STEP 3: Execute the 13 Nodes (Varies by Mode)

### Node Execution Order (Sequential)

**LAYER 1: Research (30 min)**

   - Profile audience within Saudi context
   - Map values hierarchy
   - Understand decision-making
   - Identify cultural foundations
   
2. **PR-02: Saudi Pain Point Mining** (8 min)
   - Identify specific Saudi pain points
   - Map fears and concerns
   - Identify unmet needs
   
3. **PR-03: Saudi Competitive Analysis** (8 min)
   - Map competitive landscape
   - Identify successful Saudi brands
   - Find differentiation opportunity
   
4. **PR-04: Saudi Psychology & Cultural Mapping** (4 min)
   - Deep psychology of values and motivation
   - Innovation-tradition balance
   - Islamic values integration

**LAYER 2: Strategy (30 min)**
5. **PR-05: Saudi Messaging Strategy** (10 min)
   - Define 3 core messaging pillars
   - Select 3-tier accent architecture
   - Plan Vision 2030 integration (if applicable)
   
6. **PR-06: Saudi Content Framework Selection** (8 min)
   - Choose narrative structure (story/proof/transformation/authority)

7. **PR-07: Saudi Positioning & Institutional Design** (12 min)

9. **PR-09: Saudi Body Copy** (12 min)
   - Write 3 body copy variants (different angles)

10. **PR-10: Saudi CTA Optimization** (8 min)
    - Create 3 CTA variants
    - Platform-specific formats
    - Accent tier matching
    
11. **PR-11: Saudi Objection Handling** (12 min)

    - Develop objection-specific responses
    - Provide proof for each response

**LAYER 4: Testing (20 min)**
12. **PR-12: Saudi KPI Definition** (10 min)
    - Define primary KPI
    - Define secondary KPIs
    - Measurement methodology
    - Dashboard structure
    
13. **PR-13: Saudi Testing Setup** (10 min)
    - A/B testing plan

    - Message angle testing

**LAYER 5: Integration (5 min, Throughout)**

## STEP 8: Continuous Optimization

### When Engagement Falls
1. Check platform algorithm changes (did posting time change?)
2. Review sentiment (is audience still engaged emotionally?)
3. Test new message angles (curiosity, benefit, authority - try rotation)
4. Analyze competitor activity (what changed in market?)
5. Seasonal factor (is Ramadan/Hajj affecting engagement?)

### When Conversion Drops
1. Verify CTA clarity (is action clear and specific?)
2. Check lead quality (are leads actually qualified?)
3. Test different accent tier (does Saudi vs. Gulf matter?)
4. Review objection handling (are concerns being addressed?)
5. Analyze sales process (is sales team handling leads well?)

### When Sentiment Drops
1. Review recent comments (what triggered negative sentiment?)
2. Check for competitor FUD or complaints
3. Verify compliance (any regulatory/content sensitivity issue?)
4. Assess audience perception (do followers still trust brand?)
5. Communicate (respond to concerns, clarify misunderstandings)

### When A/B Tests Underperform
1. Extend test duration (sometimes takes longer to show results)
2. Check sample size (need minimum 1,000 impressions per variant)
3. Analyze confounding factors (platform changes? seasonal factor?)
4. Try new variants (maybe message angle was wrong concept)
5. Return to control (don't force losing variant - use proven winner)

## TROUBLESHOOTING GUIDE

### Issue: Content Feels Forced or Unauthentic
**Root Cause**: Accent tier doesn't match platform/audience
**Solution**: 
1. Verify accent tier selection (Saudi/Gulf/MSA correct?)
2. Have Saudi native speakers review (authenticity check)
3. Compare to successful examples in your industry
4. Adjust language for more natural flow
5. Retest with corrected version

### Issue: Low Engagement (Below Saudi Benchmark)
**Root Cause**: Message doesn't resonate, platform timing off, or format wrong
**Solution**:
1. Check posting time (platform algorithm peak hours?)
2. Verify accent authenticity (might be turning off audience)
3. Test different message angle (curiosity vs. benefit vs. values)
4. Review platform-specific best practices (format, length, etc.)
5. Analyze competitor content (what's working for them?)

### Issue: Compliance Concerns Arise
**Root Cause**: CITC, Media Ministry, or content sensitivity issue
**Solution**:
1. Stop posting immediately if serious compliance risk
2. Document specific compliance concern
3. Consult legal/compliance team
4. Revise content to address concern
5. Resubmit for compliance review before relaunch

### Issue: Sentiment Drops Suddenly
**Root Cause**: Audience perception change, competitor attack, or content sensitivity
**Solution**:
1. Review recent comments (what triggered negative response?)
2. Check if competitor launched attack (FUD, negative comparison)
3. Verify no cultural/religious insensitivity in recent posts
4. Communicate transparently with audience (address concerns)
5. If serious: Issue clarification or apology (if warranted)

### Issue: Accent Tier Test Shows No Difference
**Root Cause**: Sample size too small or audience doesn't care about accent
**Solution**:
1. Extend test duration (need minimum 1 week per variant)
2. Increase sample size (minimum 1,000 impressions per variant)
3. Analyze platform differences (maybe accent matters MORE on TikTok)
4. Check for confounding factors (did competing post go viral?)
5. Consider: Maybe message matters more than accent tier for this product

## Advanced Tips

### 1. Build Audience Relationships, Not Just Leads
- Respond to comments authentically (in appropriate accent tier)
- Share user-generated content (show community, build trust)
- Create exclusive content for engaged followers (reward loyalty)
- Host Q&A sessions or live interactions (strengthen connection)

### 2. Leverage User-Generated Content
- Encourage customers to share their success stories (Saudi dialect authenticity)
- Create hashtag campaigns for community engagement
- Feature customer testimonials in your content
- Build social proof through real user voices (most powerful for Saudi market)

### 3. Seasonal Campaign Planning
- Plan Ramadan content 6 weeks in advance (messaging shift)
- Prepare National Day tie-ins 2 months ahead
- Build Eid content calendar during Ramadan
- Regular season content: Standard rotation with optimization tweaks

### 4. Influencer Partnerships
- Partner with Saudi entrepreneurs (micro/nano influencers often most effective)
- Verify influencer licensing (Saudi Media Ministry requirement for some)
- Align with creator's values and audience
- Co-create content that feels authentic (not forced sponsorship)

### 5. Cross-Channel Consistency
- Adapt messaging to platform (not identical across channels)
- Maintain brand voice consistency (accent tier + tone standards)
- Coordinate timing (don't launch same message on all platforms same day)
- Use platform strengths (TikTok for viral, LinkedIn for authority, Instagram for lifestyle)
